# Supply-Chain-Management-Analysis
Supply Chain Analytics Dashboard built using MySQL and Power BI to deliver data-driven insights into revenue, inventory, supplier performance, and logistics efficiency.


#Overview  
This project analyzes a fashion and beauty supply chain dataset and converts raw operational data into meaningful business insights. The focus is on understanding revenue performance, inventory status, supplier efficiency, and logistics costs through SQL analysis and an interactive Power BI dashboard.

#Dataset  
The dataset contains supply chain details such as product type, SKU, price, availability, products sold, stock levels, lead times, shipping information, supplier data, manufacturing costs, and defect rates.

#Project Steps  

Data Preparation  
Imported the dataset into MySQL

#SQL Analysis  
Wrote aggregation queries to generate KPIs  
Analyzed revenue, cost, and profit metrics  
Evaluated supplier performance and defect rates  
Identified inventory and demand patterns  

#Power BI Dashboard  
Built an interactive dashboard showing:  
Revenue and product performance  
Inventory and demand monitoring  
Supplier and logistics analysis  
Cost and profit overview  

#Key Insights  
Skincare is the top revenue-generating category  
Supplier performance varies by defect rate and margin  
Shipping mode significantly impacts total cost  
Inventory monitoring helps detect potential stock risks
